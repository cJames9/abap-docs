search help

A repository object, defined in ABAP Dictionary, that is used to create value lists. It supplies input fields on dynpros with single-column or multi-column input helps. Search helps can be associated in ABAP Dictionary with components from structures, data elements, and check tables. A search help enables searches for input values with associated data, without requiring the exact spelling of the value to be entered.
