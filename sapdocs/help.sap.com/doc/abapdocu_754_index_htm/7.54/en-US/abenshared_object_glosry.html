shared object

Instance of a shared memory-enabled class or anonymous data object that is stored in an area instance version of the shared memory. A shared object is created using the addition AREA HANDLE of statement CREATE OBJECT or CREATE DATA. References to shared objects are possible in other shared objects of the same area instance version and during the connection to the relevant area instance version using an area handle.
