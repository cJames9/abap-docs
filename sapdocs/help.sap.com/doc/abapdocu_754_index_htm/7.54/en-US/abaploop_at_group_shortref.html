LOOP AT GROUP - Quick reference

Reference

Syntax

LOOP AT GROUP group { { INTO wa }
                    | { ASSIGNING <fs> [CASTING] }
                    | { REFERENCE INTO dref }
                    | { TRANSPORTING NO FIELDS } }
                  [WHERE log_exp]
                  [GROUP BY ...].
  ...
ENDLOOP.

Effect

Reads the members of a group group from an internal table.

Additions

  • INTO, ASSIGNING, REFERENCE INTO, TRANSPORTING NO FIELDS
    Determines the output behavior (like in LOOP AT itab).

  • WHERE log_exp
    Static WHERE condition (like in LOOP AT itab).

  • GROUP BY
    Further grouping (like in LOOP AT itab).
