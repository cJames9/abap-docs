Alphabetical Overview of Catchable Runtime Errors

The following is an alphabetical list of the obsolete catchable runtime errors. For each catchable runtime error, the associated predefined exception class used to handle the corresponding runtime error is specified. If an exception class is assigned to more than one catchable runtime error, it may contain as many message texts as required. These can be found in Class Builder on the Texts tab.

Almost all the runtime errors listed here are organized together in exception groups. These exception groups are also assigned those ABAP keywords for which the runtime errors can be caught.
