Obsolete User Dialogs

Various UI technologies are available in the ABAP environment:

  • Classic SAP GUI technologies
  • Classic dynpros
  • Selection screens
  • Classic lists
  • Web-based technologies
  • Business Server Pages
  • Web Dynpro ABAP
  • SAPUI5

If possible, SAPUI5 or Web Dynpro should be used. If classic technologies are still used for any reason, the following constructs must be avoided:

  • Obsolete statements in dynpro flow logic
  • Obsolete ABAP statements for dynpros
  • Obsolete statements in list processing
  • Obsolete editor calls
