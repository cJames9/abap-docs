REFRESH CONTROL - Quick reference

Reference

Syntax

REFRESH CONTROL contrl FROM SCREEN dynnr.

Effect

Assigns the values predefined in the dynpro dynnr to components of the structure contrl defined for a table control using CONTROLS.
