Internet Transaction Server

Abbreviated as ITS. Interface between an AS ABAP and the Internet. Internet Transaction Server is based on the data interface between ABAP programs and dynpros. It enables users to communicate with AS ABAP directly from the Internet by converting the screens of dynpros to HTML files, thus enabling them to be edited in a Web browser. ITS can be installed alongside AS ABAP. Internet Communication Manager (ICM) provides AS ABAP with an interface to the Internet that is integrated directly as a process, replacing ITS.
