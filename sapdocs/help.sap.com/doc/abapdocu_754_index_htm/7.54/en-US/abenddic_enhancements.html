Enhancements

A structure, database table, database view, or domain in ABAP Dictionary created in a system can be enhanced in a different system using customizing includes, append structures, or append views without making any modifications. Customizing includes indicate points already marked by SAP for enhancements, whereas append structures can be appended anywhere.

  • Customizing includes
  • Append structures
  • Append views
  • Fixed value appends

Search helps can be expanded using append search helps.

Extensions are connection to Switch Framework whenever they are defined in a package that is assigned a switchSchalter.

Note

For CDS views, CDS view enhancements are an option for enhancements that is based on the CDS DDL in ABAP CDS.
