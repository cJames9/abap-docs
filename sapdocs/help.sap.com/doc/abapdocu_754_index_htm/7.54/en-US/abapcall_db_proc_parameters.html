CALL DATABASE PROCEDURE - parameter_list

Syntax

... [EXPORTING  p1 = a1 p2 = a2 ...]
    [IMPORTING  p1 = a1 p2 = a2 ...].

Effect

The additions EXPORTING and IMPORTING assign the actual parameters a1, a2 to the input and output parameters defined in the database procedure. This method of passing parameters can be used in both static and dynamic calls.

The proxy parameters p1, p2, ... are specified behind the additions; these are assigned the actual procedure parameters in the mapping of the database procedure proxies. The data types of the actual parameters can be determined by referencing the type declarations of the associated database procedure interfaces. The reading positions are general expression positions.
