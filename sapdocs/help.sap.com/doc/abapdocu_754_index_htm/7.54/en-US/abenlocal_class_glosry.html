local class

Class of an ABAP program that can be viewed and used statically from the ABAP program only. Local classes can be created in all program types, with the exception of interface pools and type groups. All local classes for an ABAP program are stored together in one namespace along with the local interfaces and the data types declared for the program using TYPES.
