CDS access control

Implicit restriction applied to a selection of CDS entities by access conditions defined in CDS roles when accessing objects using ABAP SQL. CDS access control is switched on by default for every CDS view and CDS table function. It can be switched off for individual views and table functions using a view annotation or table function annotation.
