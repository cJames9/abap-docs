Logical Expressions in Release 7.0, EhP2


1. Boolean functions


2. Predicate functions


3. New Boolean operator EQUIV



Note

The program DEMO_EXPRESSIONS shows examples of the logical expressions in Release 7.0, EhP2.



Modification 1

Boolean Functions

The new built-in Boolean functions boolc and boolx accept logical expressions as arguments and return their truth values in the form of character strings or byte strings.



Modification 2

Predicate Functions

The new predicate functions return truth values and can be used like relational expressions. Predicate functions for character-like arguments were introduced in Release 7.0, EhP2.



Modification 3

New Boolean Operator EQUIV

The new Boolean operator EQUIV enables an equivalence relationship between two logical expressions.
