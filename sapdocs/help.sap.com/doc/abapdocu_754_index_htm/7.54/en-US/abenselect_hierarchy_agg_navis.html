SELECT - FROM hierarchy_aggregate_navigator

Syntax

... HIERARCHY_DESCENDANTS_AGGREGATE( ... )
  | HIERARCHY_ANCESTORS_AGGREGATE( ... ) ...

Effect

Specifies a hierarchy aggregate navigator as a data source data_source in a ABAP SQL query. A hierarchy aggregate navigator accesses a hierarchy and uses aggregate functions to evaluate it. The following hierarchy aggregate navigators are available:

  • HIERARCHY_DESCENDANTS_AGGREGATE
  • HIERARCHY_ANCESTORS_AGGREGATE
