shared lock

Lock that allows other users to set further shared locks but no simultaneous exclusive locks for the locked data. The ABAP SQL statement SELECT, without the addition FOR UPDATE, and the corresponding Native SQL statement or a SELECT statement in an AMDP method do not set a corresponding database lock for the rows in question by default. A shared lock is set as an SAP lock using a call of a lock function module with the appropriate parameters.
