Description Functions for Character-Like Arguments

These functions return a numeric result.

  • charlen, dbmaxlen, numofchar, strlen - Length Functions
  • char_off - Length Function
  • find, find_... - Search Functions
  • count, count_... - Search Functions
  • distance - Distance Function

Note

These functions can be used in suitable operand positions, for example directly as an operand of arithmetic or relational expressions.
