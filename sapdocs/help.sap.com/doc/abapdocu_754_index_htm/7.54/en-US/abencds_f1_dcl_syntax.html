ABAP CDS - DCL Statements

The data control language CDS DCL in ABAP CDS follows the general CDS syntax rules and comprises the following statements for the definition of CDS roles and CDS access policies in DCL source code:

  • DEFINE ROLE
  • DEFINE ACCESSPOLICY

Notes

  • The DCL source code for CDS access control has its own editor in ADT, which is documented there.
  • The names of the items of DCL source code for CDS roles and CDS access policies are in the same namespace.
  • DCL source code can also be displayed in Repository Browser in ABAP Workbench.
