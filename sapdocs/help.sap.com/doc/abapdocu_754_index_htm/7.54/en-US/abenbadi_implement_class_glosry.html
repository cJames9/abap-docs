BAdI implementation class

Global class that implements one or more BAdI interfaces and therefore also their BAdI methods. BAdI implementation classes are the main components of BAdI implementations, and their instances act as object plug-ins for functional enhancements of ABAP programs. A BAdI implementation class that is not part of a BAdI implementation is not linked to an enhancement implementation. The latter can be used, for example, as fallback BAdI implementation classes of BAdIs.
