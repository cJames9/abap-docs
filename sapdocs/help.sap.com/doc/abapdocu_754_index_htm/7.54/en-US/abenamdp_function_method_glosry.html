AMDP function implementation

AMDP method with return value indicated by BY DATABASE FUNCTION in the implementation part of the class. AMDP functions are implemented in an AMDP function implementation. An AMDP function implementation for an AMDP scalar function can be called in ABAP programs in the same way as a regular functional method. An AMDP function implementation for an AMDP table function, on the other hand, is generally used to implement a CDS table function. It can be used as the data source of ABAP SQL read statements using the CDS entity, but it cannot be called in ABAP programs. See also AMDP procedure implementation.
