ABAP SQL - Built-In Functions sql_func

The following built-in database functions managed by ABAP Dictionary can either be specified in ABAP SQL as SQL expressions, in all places where these are possible:

  • SQL functions
  • Special functions

Note

Certain SQL functions can be executed in the table buffer. If any other built-in functions are used, table buffering is bypassed.
