CDS role

A CDS object defined using the statement DEFINE ROLE in DCL source code in CDS DCL in ABAP CDS. This object applies CDS access control based on access conditions when non-abstract CDS entities are accessed using ABAP SQL. Currently, a CDS role is assigned to every user. The access conditions of a CDS role evaluate the classic authorizations of the current user.
