ABAP SQL - SQL Conditions sql_cond

Syntax

...   rel_exp
    | [NOT] sql_cond [AND|OR sql_cond] ...

Effect

Logical expression sql_cond for formulating a condition in ABAP SQL. A logical expression consists of one or more relational expressions rel_exp that are joined by AND or OR and can be negated with NOT. There are different types of relational expressions:

  • Relational expressions for statements
  • Relational expressions for expressions
