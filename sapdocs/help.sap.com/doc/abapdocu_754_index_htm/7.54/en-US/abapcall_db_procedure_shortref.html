CALL DATABASE PROCEDURE - Quick reference

Reference

Syntax

CALL DATABASE PROCEDURE proxy|(prox_name)
    [CONNECTION con|(con_syntax)]
    { [EXPORTING  p1 = a1 p2 = a2 ...]
      [IMPORTING  p1 = a1 p2 = a2 ...] }
  | { [PARAMETER-TABLE ptab] }.

Effect

Calls a database procedure of SAP HANA database written in SQLScript using a database proxy proxy.

Additions

  • CONNECTION con|(con_syntax)
    Executes the statement on a secondary connection.

  • EXPORTING p1 FROM a1 p2 FROM a2 ...
    Passes actual parameters a1, a2, ... to input parameters p1, p2, ...

  • IMPORTING p1 TO a1 p2 TO a2 ...
    Inherits output parameters p1, p2, ... in actual parameters a1, a2, ...

  • PARAMETER-TABLE ptab
    Assigns dynamic actual parameters to formal parameters using an internal table ptab of type ABAP_ PARMBIND_TAB.
