exception class

Special class that is the basis for handleable exceptions. If an exception is raised, an object of an exception class is raised. There are predefined exception classes for exceptions of the runtime environment, and self-defined exception classes for custom applications. The superclasses for all exception classes are CX_STATIC_CHECK, CX_DYNAMIC_CHECK, or CX_NO_CHECK. These are used to divide exceptions into exception categories.
