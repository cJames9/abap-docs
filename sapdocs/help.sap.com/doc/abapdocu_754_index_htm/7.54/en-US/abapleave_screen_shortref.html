LEAVE SCREEN - Quick reference

Reference

Syntax

LEAVE { SCREEN | {TO SCREEN dynnr} }.

Effect

Exits the current dynpro and continues processing using the next dynpro.

Addition

  • TO SCREEN dynnr
    Specifies the next dynpro in dynnr.
