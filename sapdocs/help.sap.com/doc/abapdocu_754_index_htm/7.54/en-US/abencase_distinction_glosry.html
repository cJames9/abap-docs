case distinction

Control structure with the control statement CASE or CASE TYPE OF, condition expression with the conditional operator SWITCH, CASE expression in ABAP SQL, or CASE expression in CDS DDL. In case distinctions, the value of an operand determines which one of multiple source code sections is executed.
