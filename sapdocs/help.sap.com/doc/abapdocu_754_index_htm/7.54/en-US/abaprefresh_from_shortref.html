REFRESH - FROM - Quick reference

Reference

Syntax

REFRESH itab FROM TABLE dbtab.

Effect

Obsolete: Deletes all rows of an internal table itab and fills the table with rows from the database table or view dbtab. The content of the table work area dbtab is used as the search key.
