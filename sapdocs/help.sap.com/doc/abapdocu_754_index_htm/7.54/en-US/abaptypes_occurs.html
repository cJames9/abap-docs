TYPES - OCCURS

Quick Reference

Obsolete Syntax

TYPES dtype { {TYPE [REF TO] type}
            | {LIKE [REF TO] dobj} } OCCURS n.

Effect

This statement is forbidden in classes. The statement has the same function as the following TYPES statement for defining standard table types, and is replaced by the following:

TYPES dtype { {TYPE STANDARD TABLE OF [REF TO] type}
            | {LIKE STANDARD TABLE OF [REF TO] dobj} }
            WITH NON-UNIQUE DEFAULT KEY
            INITIAL SIZE n.
