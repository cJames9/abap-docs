arith_exp - Quick reference

Syntax

... { [+|-] operand1
      [{+|-|*|/|DIV|MOD|**} [+|-] operand2
      [{+|-|*|/|DIV|MOD|**} [+|-] operand3
      ... ]] } ...

Effect

Arithmetic expression for calculating a numeric result from the operand operand. Arithmetic expressions can be parenthesized explicitly using ( ) and can occur in the reading positions of certain statements.

Additions

  • +|-
    Sign of an operand operand.

  • +|-|*|/|DIV|MOD|**
    Arithmetic operators.
