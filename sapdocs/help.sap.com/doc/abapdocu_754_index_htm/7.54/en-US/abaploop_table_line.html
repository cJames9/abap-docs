LOOP AT itab - TABLE LINE

Obsolete Syntax

LOOP AT itab ... WHERE TABLE LINE ....

Effect

The addition TABLE LINE can also be specified outside of classes in the WHERE condition of a LOOP statement, instead of the pseudo component table_line.

Note

ABAP Compiler should consider this addition as an error, retained only for reasons of downward compatibility. Always specify the pseudo component table_line instead of TABLE LINE.

Bad Example

DATA: itab TYPE TABLE OF i,
      wa   TYPE i.

LOOP AT itab INTO wa WHERE TABLE LINE > 10.

ENDLOOP.

Good Example

DATA: itab TYPE TABLE OF i,
      wa   TYPE i.

LOOP AT itab INTO wa WHERE table_line > 10.

ENDLOOP.
