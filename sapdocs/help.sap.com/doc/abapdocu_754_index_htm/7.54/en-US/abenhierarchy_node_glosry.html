hierarchy node

Row of a hierarchy in ABAP SQL. A hierarchy node contains special hierarchy columns for the hierarchy attributes of the hierarchy. Hierarchy nodes can be root nodes, parent nodes, child nodes, leaf nodes, orphan nodes, sibling nodes, descendant nodes, and ancestor nodes.
