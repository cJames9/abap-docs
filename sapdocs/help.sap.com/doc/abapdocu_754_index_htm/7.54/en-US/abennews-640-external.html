External Interfaces in Release 6.40


1. New callback routine in asynchronous RFC


2. Simple Transformation calls


3. Control options for transformations



Modification 1

New Callback Routine in Asynchronous RFC

In asynchronous RFC, the addition CALLING addition was introduced to also specify methods as callback routines.

Note

This change was also transported back to Release 6.20.



Modification 2

Simple Transformation Calls

CALL TRANSFORMATION can now be used to call Simple Transformations, as well as XSLT programs.



Modification 3

Control Options for Transformations

In CALL TRANSFORMATION, the addition OPTIONS can be used to specify control options for the called transformations.
