dialog box

Also called a modal dialog box. Display of the GUI window of a dynpro sequence called using CALL SCREEN, where the previous GUI window remains visible, but inactive. Up to nine dialog boxes can be stacked over a GUI window (see popup level). Modeless dialog boxes, where the previous GUI window remains active, can be implemented in SAP GUI only as GUI controls.
