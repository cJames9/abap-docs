byte order

Determines the order in which the binary representation of the data types i, int8, decfloat16, decfloat34, f, s, or a character is stored in the memory. A distinction is made between big and little endian. In the former case, the system writes the byte with the highest level, and in the second case the byte with the lowest level, to the first position in the memory.
