Unicode

Language-independent character set for international data processing (www.unicode.org). Subset of the UCS character set in accordance with ISO-10646. Unicode can represent 1114111 different characters numbered as hexadecimal numbers from U+0000 to U+10FFFF. The Unicode character set is normally accessed using a UTF or UCS character representation.
