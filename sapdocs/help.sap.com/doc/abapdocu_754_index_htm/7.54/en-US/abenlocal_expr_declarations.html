Local Declarations in Expressions

In most constructor expressions, variables and field symbols can be used as local helper fields. They are defined using the language element LET in a

  • LET expression let_exp

LET expressions can be used in the following constructor expressions:

  • NEW
  • Single values
  • Structures
  • Internal tables
  • Classes
  • VALUE
  • Structures
  • Internal tables
  • CONV
  • CAST
  • EXACT
  • REDUCE
  • COND
  • SWITCH

LET expressions are also possible in all iteration expressions with FOR.
