subroutine

Obsolete procedure that can be defined in any ABAP program (except type pools, class pools, and interface pools), but can only be implemented outside classes between the statements FORM and ENDFORM. The statement PERFORM can be used to call the subroutines of an ABAP program and of any other programs.
