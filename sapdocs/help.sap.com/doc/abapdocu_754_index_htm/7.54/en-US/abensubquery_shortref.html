subquery - Quick reference

Syntax

... ( SELECT { ...
               FROM ... }
           | { FROM ...
               FIELDS ... }
             [WHERE sql_cond]
             [GROUP BY ...] [HAVING sql_cond] ) ...

Effect

Parenthesized SELECT statement that can be used in conditions sql_cond of ABAP SQL statements, in common table expressions of the statement WITH, or as a data source of an INSERT statement for a subquery.

Additions

  • As in the statement SELECT.

  • The additions SINGLE, ORDER BY, and INTO cannot be used.

  • Further additions may also be excluded, depending on how they are used.
