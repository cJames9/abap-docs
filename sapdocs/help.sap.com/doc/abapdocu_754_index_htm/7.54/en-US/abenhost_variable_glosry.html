host variable

ABAP data object specified in an operand position of an ABAP SQL statement or of a statically embedded Native SQL statement . The ABAP data object can be a variable or a constant (see host constant). In ABAP SQL, a host variable must be prefixed with the escape character @ ; in Native SQL, with the escape character : . In ADBC, the placeholder ? (to which ABAP data objects are bound) is used instead of host variables.
