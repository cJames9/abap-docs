reverse - Reverse Function

Syntax

... reverse( [val =] text  ) ...

Effect

This function returns a character string that reverses the content of text.

The return code has the type string.

Example

Gets the Latin alphabet in reverse.

DATA(result) = reverse( sy-abcde ).
